# Visual Studio Code

Visual Studio Code，也被很多人称作VSCode，想必已经是家喻户晓。这个编辑器基本上对于所有的语言都通吃，而且还有非常丰富的插件支持，只不过可能专门面向Python的优化少了些，比如Python中一些函数的详细信息和语法提示等，以及代码格式提醒。不过总体来说依然是一款不错的编辑器。当然，资源占用也不多，可以说是老电脑福音。
